# Korean Education Law Ontology Visualization

MiroFish GraphRAG + Multi-Agent Simulation Architecture

Interactive visualization of Korean education law hierarchy with multi-agent policy simulation.

## Features
- Ontology Graph: 27 law nodes, 35 edges, D3.js force-directed layout
- Multi-Agent Simulation: 6 stakeholder agents
- Policy Scenarios: 6 education policy scenarios
- Law-Project Matrix: 27 laws x 9 project areas

## Tech Stack
React 18 + D3.js v7 + Vite
